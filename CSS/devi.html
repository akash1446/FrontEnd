Diagonal Color with CSS Linear Gradient. The linear-gradient function lets you blend colors in any direction—including diagonally. background-image: linear-gradient(direction, color1, color2); To make the transition go diagonally (like from top-left to bottom-right), use to bottom right background-image: linear-gradient(to bottom right, red, yellow) This starts with red at the top-left and blends to yellow at the bottom-right. You can also use an angle. background-image: linear-gradient(45deg, red, yellow); You can add more colors/stops for complex effects. Select the cell. 2.Right-click > Format Cells > Border, and pick a diagonal border (to create the dividing line). 3.
